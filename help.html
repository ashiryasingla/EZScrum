Need Help Navigating?

Follow these steps:



  1. You can add new sprints using the "New Sprint" button..
  2. Write the Duration of your sprint in days
  3. You can add new user stories using the "Add item" button in the Product backlog.
  4. rename the user story and click the green button to shift it to the active sprint.
  5. Assign the user story to the respective member .
  6. You can remove user stories using the "Remove Item" button in the Product backlog.
  7. You can remove sprints by pressing the small red button on the sprint box.
  8. You can remove user stories from a sprint by clicking "Remove Item".



If you're still confused, ask a teammate or reach out via the Contact page!
